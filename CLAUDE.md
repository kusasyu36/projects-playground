# Projects 共通ルール

## 開発ガイドライン
プロジェクト開始・再開時は、必ず以下を参照すること：
- /Users/s-kusaba/projects/_guides/サービス開発フレームワーク.md
- /Users/s-kusaba/projects/_guides/コンテキスト管理ガイド.md

## セッション開始時（自動実行されるHookの出力を確認）

1. **バージョン確認**: 新バージョンがある場合
   - ユーザーに「更新しますか？」と確認
   - OKなら `npm update -g @anthropic-ai/claude-code` を実行
   - 更新後、新機能を**非エンジニアにもわかるように**説明

2. **進捗確認**: `進捗.md` の内容を要約して報告
   - なければ作成を提案

3. **ガイドライン確認**: _guidesに沿って作業を進める

## 保存ルール
- 作業の区切りで `進捗.md` を更新
- 動作確認後はgit commit

## 自動保存（Hooks有効）

| タイミング | 動作 |
|-----------|------|
| ファイル編集後 | `git add -A`（自動ステージング） |
| 毎応答後 | `git commit`（自動コミット） |

※ ~/projects/ 配下でのみ有効
